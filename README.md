# 📩 Spam Email Detection Web App

A Machine Learning web application that classifies messages as **Spam or Not Spam (Ham)** using Natural Language Processing (NLP) and a Naive Bayes model. Built with **Streamlit** for an interactive user interface.

---


## 📌 Project Overview

This project detects whether a given message is spam or not spam using machine learning. It converts text into numerical features using **TF-IDF Vectorization** and applies a **Multinomial Naive Bayes classifier** for prediction.

---

## 🧠 Machine Learning Pipeline

1. Load dataset (SMS spam dataset)
2. Text preprocessing
3. TF-IDF Vectorization
4. Train-test split
5. Train Naive Bayes model
6. Evaluate model
7. Deploy using Streamlit

---

## 🛠️ Technologies Used

- Python
- Pandas

- Streamlit
- NLP (TF-IDF Vectorizer)
- Naive Bayes Classifier
